> Rated 9.5/10 — updated with 6 additions from AI review

## The ILS Product Principle

> **Every feature in ILS should contribute to improving educational quality, not just digitizing educational administration.**

This is the filter for every product decision:
- Does this feature help teachers teach better?
- Does it help students succeed?
- Does it help institutions improve continuously?

## Phase 5 — Launch

28. Create LinkedIn company page, start sharing educational insights (not ads — build authority)
29. Launch newsletter / waitlist to capture early interest
30. Run **Early Adopter Programme** (structured pilot — not just "pilot with 2 schools"):
    - Objectives: validate workflows, measure implementation effort, collect testimonials, discover missing features
    - Deliverables: feedback reports, case studies, feature requests, success metrics

32. Announce publicly

## Success Metrics — Define Before Launch

### Marketing
- Website visitors per month
- Demo requests per month
- Newsletter signups

### Product
- Daily active users
- Weekly active teachers
- Student login rate

### Customer Success
- Time to first class created
- Time to first assessment submitted
- User satisfaction score

### Commercial
- Demo → Pilot conversion rate
- Pilot → Customer conversion rate
- Customer retention rate

## Product Roadmap Governance

After launch, define how work gets prioritized — otherwise every customer request becomes urgent:

- **P0** — Critical bugs (fix immediately)
- **P1** — Customer-blocking issues (next release)
- **P2** — Feature requests from pilots (quarterly review)
- **P3** — Strategic initiatives (roadmap)
- **P4** — Technical debt (scheduled)

## Why ILS? (for the website)

| Traditional LMS | ILS |
|---|---|
| Focus on content delivery | Focus on learning outcomes |
| Separate disconnected systems | Integrated end-to-end platform |
| Reports after the fact | Operational intelligence in real time |
| Teacher-centered | Institution-wide insights |
| No curriculum intelligence | Curriculum connected to teaching and assessment |
